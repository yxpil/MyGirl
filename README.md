# 滢河 - 基于RNN的聊天机器人

## 项目介绍

本项目使用循环神经网络（RNN）构建中文聊天机器人，采用GRU架构实现语言模型。

**技术亮点：**
- ✅ 训练数据规模：65万条中文对话
- ✅ INT8动态量化，模型体积减小40%
- ✅ 仅用朴素RNN架构实现自然闲聊效果
- ✅ CPU/GPU 跨平台兼容运行

## 核心创新

### 1. 中文专属分词算法（完全自研）
- 采用 **最长优先匹配 + N-gram统计** 的中文专属分词
- 按字频、词频双重统计构建 **21万超大词表**
- 设计思想：用更大的词表减少序列长度，**显著节约RNN运算量**
- 从根本上解决了BPE不适合中文语义的问题

### 2. 模型架构
- 模型类型：RNN Language Model (3层GRU)
- 隐藏层维度：512
- 序列长度：256
- 词表大小：209,958 tokens

### 3. 部署优化
- INT8动态量化，推理速度提升，体积减小40%
- CPU/GPU双平台自动适配

## 环境要求

- Python 3.x
- PyTorch 2.0+
- CUDA (推荐)

## 安装依赖

```bash
pip install -r requirements.txt
```

## 启动方式

```bash
python chat.py
```

## 使用说明

1. 运行程序后直接输入文字进行对话
2. 输入 `quit`, `exit`, `q` 或 `退出` 结束程序
3. 输入 `temp=0.4` 可以调整生成温度（默认0.4）
   - 温度越低，生成结果越确定
   - 温度越高，生成结果越多样化
